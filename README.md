# Monster Hunter Rise: Cold/Hot Drink Mechanic Mod

Цей мод додає механіку холодних та гарячих напоїв до Monster Hunter Rise, створюючи додатковий рівень стратегії та реалізму в грі.

## Особливості

### Температурні Ефекти
- **Гарячі зони** (Sandy Plains, Flooded Forest, частини Lava Caverns):
  - Поступова втрата витривалості
  - Повідомлення про спеку
  - Потреба у прохолодних напоях

- **Холодні зони** (Frost Islands, холодні частини Lava Caverns):
  - Сповільнена регенерація здоров'я
  - Повідомлення про холод
  - Потреба у гарячих напоях

- **Нормальні зони** (Shrine Ruins):
  - Без температурних ефектів

### Система Напоїв
- **Cool Drink (ID: 79)**: Захищає від жари протягом 5 хвилин
- **Hot Drink (ID: 80)**: Захищає від холоду протягом 5 хвилин
- Напої працюють тільки у відповідних температурних умовах
- Таймер захисту відображається в інтерфейсі мода

### Інтерфейс
- Інтерактивне меню мода через REFramework
- Відображення поточного температурного стану
- Статус захисту та час, що залишився
- Кнопки для тестування напоїв
- Мапа температур для всіх зон

## Встановлення

### Вимоги
1. Monster Hunter Rise (Steam версія)
2. [REFramework](https://github.com/praydog/REFramework) встановлений
3. REFramework Console включений в налаштуваннях

### Кроки встановлення
1. Завантажте `cold_hot_drink_mod.lua`
2. Скопіюйте файл до папки:
   ```
   [Monster Hunter Rise Install Directory]/reframework/autorun/
   ```
3. Запустіть гру
4. Мод буде автоматично завантажений при старті

### Перевірка встановлення
1. Натисніть Insert у грі для відкриття REFramework меню
2. Знайдіть розділ "Cold_Hot_Drink_Mod v1.0.0"
3. Якщо розділ присутній, мод успішно встановлений

## Як використовувати

### Початок квесту
- Мод автоматично ініціалізується при старті квесту
- З'явиться повідомлення про ініціалізацію температурної системи

### У грі
1. **Перевірка температури**: Відкрийте меню мода для перевірки поточної температури
2. **Використання напоїв**: 
   - У гарячих зонах використовуйте Cool Drink
   - У холодних зонах використовуйте Hot Drink
   - Напої не діють у неправильних умовах
3. **Моніторинг ефектів**: Слідкуйте за повідомленнями про температурні ефекти

### Температурна карта зон
- **Зона 1** (Shrine Ruins): Нормальна температура
- **Зона 2** (Sandy Plains): Жарка
- **Зона 3** (Frost Islands): Холодна  
- **Зона 4** (Flooded Forest): Жарка (вологість)
- **Зона 5** (Lava Caverns - холодні частини): Холодна
- **Зона 6** (Lava Caverns - гарячі частини): Жарка

## Налаштування

### Редагування коду
Ви можете змінити наступні параметри у файлі мода:

```lua
-- Тривалість ефекту напоїв (у секундах)
local DRINK_DURATION = 300 -- 5 хвилин

-- Інтервал перевірки температури
local TEMP_CHECK_INTERVAL = 5 -- кожні 5 секунд

-- ID предметів
local COLD_DRINK_ID = 79  -- Cool Drink
local HOT_DRINK_ID = 80   -- Hot Drink
```

### Налаштування температур зон
Змініть таблицю `area_temperatures` для налаштування температур різних зон:

```lua
local area_temperatures = {
    [1] = TEMP_NORMAL,  -- Shrine Ruins
    [2] = TEMP_HOT,     -- Sandy Plains
    [3] = TEMP_COLD,    -- Frost Islands
    -- додайте або змініть зони за потребою
}
```

## Усунення неполадок

### Мод не завантажується
1. Переконайтеся, що REFramework правильно встановлений
2. Перевірте, чи файл знаходиться у правильній папці `autorun`
3. Перезапустіть гру

### Напої не працюють
1. Переконайтеся, що ви у правильній температурній зоні
2. Перевірте, чи правильні ID предметів у коді
3. Використовуйте тестові кнопки в меню мода

### Температурні ефекти не застосовуються
1. Перевірте, чи квест активний
2. Переконайтеся, що ви не захищені напоєм
3. Перевірте консоль REFramework на помилки
